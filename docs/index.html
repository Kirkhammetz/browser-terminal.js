Welcome to browser-terminal.js

Simple Vanilla Javascript Terminal Emulator, with no depency.

- Why? If you need to simulate a CLI tool on your documentation and you want to make it interactive

- You just want to play around

- You want a base to start without reinventing the wheel.

It's simple, it's extensible, it's lightwell


* Type help for list of commands
* Type man <command> for extra info
